## :rocket: Getting started
depend via Maven:
```xml
<dependency>
  <groupId>run.cobalt</groupId>
  <artifactId>vortex</artifactId>
  <version>0.0.2</version>
</dependency>
```
or Gradle:
```kotlin
implementation("run.cobalt:vortex:0.0.2")
```
